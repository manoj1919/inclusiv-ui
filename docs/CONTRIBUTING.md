# Contributing

The project is in pre-launch alpha and not yet open for external contributions. This file is a placeholder; once the data schema and pipeline stabilize, contribution guidance will land here.

## Reporting data errors

If you spot a number that looks wrong — particularly if you're a parent, teacher, or special ed administrator with direct knowledge — please [open an issue](https://github.com/manoj1919/asd-schools-ca/issues) with:

- The district and field in question
- The current displayed value
- What you believe the correct value should be
- A source link if possible

## Areas where we'll eventually want help

- SELPA annual report ingestion (varies wildly by SELPA)
- Plain-language explainer copy reviewed by educators
- Accessibility audit of the frontend
- Translation (Spanish first; large CA population)

## Code of conduct

Treat everyone with respect. This project exists for families of children with disabilities — many contributors and readers are dealing with hard situations. Default to kindness.
